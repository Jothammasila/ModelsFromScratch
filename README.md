# ModelsFromScratch

This repository contains multiple machine learning models implemented **from scratch** using Python.  
Each model is implemented without relying on high-level ML libraries, focusing on understanding the algorithms and their inner workings.

## 🚀 Projects

### 1. Logistic Regression
- Folder: `Logistic Regression/`
- Description: Implements binary logistic regression from scratch using gradient descent.
- Key Files:
    -   `data_generator.py` – Generates random binary classification dataset (the data is saved in `data.csv` file)
    - `train.py` – Training script
    - `LogisticRegressor.py` – Implementation of Logistic Regression class
    - `normalization.py` - Implementation of the data normalization function.
    - `model.py` - Model training script (the model is saved in a `logistic_regressor_model.pkl`)
    - `test_model.py` - Loading and testing the trained model


*(Future models will be added here in their own folders.)*
